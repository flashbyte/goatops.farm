goatops.farm

An API inspired by goatops.com to programmatically obtain your daily dose of facts about farm creatures so you can integrate them into the tools of your ops day job until you can finally start farming.

Usage

Get all facts about a farm creature: 

curl https://goatops.farm/api/v1/facts

Get random facts about farm creatures: 

curl https://goatops.farm/api/v1/facts/random

See swagger-ui for detailed API endpoint docs.
