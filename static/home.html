goatops.farm

An API inspired by goatops.com to programmatically obtain your daily dose of facts about farm creatures so you can integrate them into the tools of your ops day job until you can finally start farming.

Usage

List all farm creatures and facts about them: 

curl https://goatops.farm/api/v1/creatures

Get random facts about a farm creature: 

curl https://goatops.farm/api/v1/creatures/goat/random-facts

See swagger-ui for detailed API endpoint docs.

Source

Source code on GitHub. If you want to see more creatures and facts send a PR!
